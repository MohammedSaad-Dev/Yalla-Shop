#  Yalla Shop - Graduation Project 2026

**Yalla Shop** is a modern, high-performance E-Commerce platform designed to deliver a seamless shopping experience. This project serves as my final graduation project, focusing on a robust backend architecture, secure authentication, scalable database design, and real-time payment handling.

---

##  Architecture & Technical Stack

The system is built using a decoupled architecture, leveraging a secure and powerful backend API designed to handle complex business logic, enterprise-grade inventory tracking, and high-concurrency operations.

* **Backend Framework:** `.NET Core API` (C#)
* **Database Management:** `Microsoft SQL Server` & `SQL Server Management Studio (SSMS)`
* **ORM (Data Access):** `Entity Framework Core` (Code-First approach with optimized database relationships)
* **Security & Auth:** `JWT (JSON Web Tokens)` with role-based authorization (Admin / Vendor / Customer)
* **API Documentation:** `Swagger UI` for comprehensive, interactive endpoint testing

---

##  Key Features

-   **Secure Authentication:** Complete registration and login system backed by JWT, including token refreshing and password hashing.
-   **Advanced Inventory Management:** Specialized inventory control designed for scalable commodity and product handling.
-   **Interactive Shopping Cart:** Real-time adding, updating, and removing of products with strict validation against stock levels.
-   **Secure Checkout & Payments:** Fully integrated, secure payment processing modules to handle transactions safely.
-   **Dynamic API Documentation:** A fully documented Swagger interface for seamless backend-frontend integration.
